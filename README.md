# Causal-Explainable-Predictive-Maintenance-NASA-CMAPSS
A research project exploring explainable and causal AI methods for predictive maintenance using NASA C-MAPSS turbofan engine degradation data.
Dataset:
NASA C-MAPSS Turbofan Engine Degradation Simulation Dataset

## Citation:
A. Saxena and K. Goebel (2008).
"Turbofan Engine Degradation Simulation Data Set"
NASA Ames Research Center.
