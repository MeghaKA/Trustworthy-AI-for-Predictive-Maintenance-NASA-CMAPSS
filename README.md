# Trustworthy Causal Artificial Intelligence for Predictive Maintenance using NASA C-MAPSS

---

## Overview

This repository presents a research-oriented **Trustworthy Causal Artificial Intelligence (TCAI)** framework for industrial predictive maintenance using the **NASA C-MAPSS turbofan engine degradation dataset**.

The project was developed as an independent research work and is accompanied by a comprehensive technical report documenting the complete methodology, implementation, experiments, causal analysis, discussion, and future research directions.

Unlike conventional Remaining Useful Life (RUL) prediction systems that stop at forecasting equipment failure, this framework integrates:

- Predictive Machine Learning
- Explainable Artificial Intelligence (XAI)
- Temporal Causal Discovery
- Structural Causal Modelling (SCM)
- Structural Equation Modelling (SEM)
- Counterfactual Reasoning
- Maintenance Decision Support

to build transparent, interpretable, and trustworthy industrial AI systems.

The proposed research demonstrates the transition from

**Prediction → Explanation → Causal Understanding → Decision Support**

# Research Objectives

This research investigates how industrial AI systems can move beyond accurate prediction to answer practical engineering questions such as:

- When will the engine fail?
- Why is the engine degrading?
- Which sensors are true degradation drivers?
- Which variables are causally related rather than merely correlated?
- What maintenance intervention could improve Remaining Useful Life?
- How can AI support trustworthy engineering decisions?

# Dataset

**NASA Commercial Modular Aero-Propulsion System Simulation (C-MAPSS)**

Dataset Used

- FD001 Training Dataset
- FD001 Testing Dataset
- Remaining Useful Life (RUL) Labels

Dataset Characteristics

- 100 Training Engines
- 100 Testing Engines
- 21 Sensor Measurements
- Operating Conditions
- Complete Engine Degradation Trajectories

https://ti.arc.nasa.gov/tech/dash/groups/pcoe/prognostic-data-repository/
